## ğŸ†˜ Troubleshooting

### Issue: Site not loading or showing old version

**Solution:**
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+F5 or Cmd+Shift+R)
3. Try incognito/private mode
4. Check Firebase Console for deployment status

### Issue: Google Drive OAuth not working

**Solution:**
1. Verify redirect URIs in Google Cloud Console
2. Make sure you added: `https://charleswebeditor.web.app`
3. Wait 5-10 minutes after updating OAuth settings
4. Clear browser cookies and try again

### Issue: Some features not working

**Solution:**
1. Open browser console (F12)
2. Look for JavaScript errors
3. Take screenshot and check documentation
4. Try different browser

### Issue: Need to rollback deployment

**Solution:**
```bash
cd ChassWED2
npx firebase hosting:rollback
```

This instantly reverts to the previous version.
