# Archive Directory

This directory contains unused code and test files that have been archived to keep the main codebase clean.

## Structure

### `unused-code/`
Contains unused editor implementations that are not currently in use:
- `ChassWED.js` - Alternative monolithic editor implementation (unused)
- `main-new.js` - Entry point for modular architecture (unused)
- `src/` - Complete modular architecture implementation (unused)

**Note:** The main application currently uses `public/main.js` (monolithic implementation).

### `tests/`
Contains test files and diagnostic scripts:
- Various test HTML/JS files
- Diagnostic scripts for icon loading
- Template verification scripts

**Note:** These files are kept for reference but are not part of the active codebase.
